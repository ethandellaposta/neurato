# /task - Quick Task Commands

Simple task management commands without prefixes.

## Usage

```bash
# Pin a task
/task pin "Your task description"

# Show pinned tasks
/task show

# Remove a pinned task
/task unpin <task-id>

# Execute all pinned tasks
/task pop

# Show task queue
/task queue
```

## Commands

### /task pin
Pin a task to your queue for later execution.

### /task show
List all currently pinned tasks.

### /task unpin
Remove a specific pinned task by ID.

### /task pop
Execute and clear all pinned tasks.

### /task queue
Show task queue status and backlog.

This provides quick task access without prefixes.
