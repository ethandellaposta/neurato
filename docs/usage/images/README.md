# Usage Screenshot Guide

Add usage screenshots in this folder so links in usage docs remain stable.

## Expected files

- `launch-debug.png` — app opened from Debug build
- `timeline-overview.png` — timeline with at least one track/clip
- `mixer-panel.png` — mixer visible (`Cmd+M`)
- `piano-roll.png` — MIDI editor open
- `audio-clip-editor.png` — audio clip editor open
- `offline-bounce.png` — bounce/export flow visible

## Capture notes

- Use native resolution when practical.
- Prefer clean sample projects with readable labels.
- Avoid personal/private file paths in screenshots.
- Keep filenames exactly as listed above.
